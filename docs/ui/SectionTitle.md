# SectionTitle

Small, medium-weight section heading label used to introduce a block of content within a page (not a settings row — see [SettingsSectionHeader](SettingsSectionHeader.md) for that).

Source: [`SectionTitle.tsx`](../../src/app/components/ui/SectionTitle.tsx)

## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `children` | `React.ReactNode` | — | Heading text/content |

## Usage

```tsx
import { SectionTitle } from "../ui";

<SectionTitle>Recent Licenses</SectionTitle>
```
